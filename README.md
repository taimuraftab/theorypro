# TheoryPro

A modern full-stack quiz and learning platform designed to deliver structured knowledge through category-based assessments and explanation-driven learning.

TheoryPro focuses on performance, scalability, and clean user experience while maintaining a simple and maintainable architecture.

---

## Overview

TheoryPro is built using a React frontend and a Node.js + Express backend, powered by MongoDB for persistent data storage.  
The application dynamically serves quiz content and categories directly from the database, allowing easy content scaling.
